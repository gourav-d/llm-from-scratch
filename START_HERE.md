## ğŸ†˜ Common First-Day Issues

### Issue: "python is not recognized"
**Solution:** Python not in PATH
1. Find Python installation
2. Add to system PATH
3. Restart terminal
4. See QUICK_START.md for details

### Issue: "No module named X"
**Solution:** Install dependencies
```bash
pip install -r requirements.txt
```

### Issue: "Can't activate venv"
**Solution:** Try:
```bash
venv\Scripts\activate.bat
```

### Issue: "IndentationError"
**Solution:** Python is sensitive to indentation
- Use 4 spaces per level
- Don't mix tabs and spaces
- VS Code helps with this!

## ğŸ“ Learning Tips from Claude

### For .NET Developers:

**What's Similar:**
- Variables, functions, classes
- if/else, loops, logic
- OOP concepts
- Same programming thinking

**What's Different:**
- No type declarations
- No semicolons
- Indentation matters!
- snake_case not camelCase
- Different keywords (and, or, not)

**How to Learn Fast:**
1. **Stop comparing** - Python is Python, not C#
2. **Embrace differences** - They make Python powerful
3. **Type everything** - Don't copy-paste
4. **Experiment** - Break things on purpose
5. **Ask Claude** - I'm here to help!
