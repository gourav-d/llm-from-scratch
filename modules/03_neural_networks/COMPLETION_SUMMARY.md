## 💡 Key Concepts You'll Master

### Core Understanding
✅ **How neurons compute:** `z = w·x + b`, then `activation(z)`
✅ **Why activation matters:** Without it, deep = shallow (no learning complex patterns)
✅ **How learning works:** Backpropagation traces errors backward
✅ **How weights update:** Gradient descent adjusts weights to minimize loss

### Practical Knowledge
✅ **Which activation to use:**
- Hidden layers: ReLU (or GELU for transformers)
- Binary classification: Sigmoid
- Multi-class: Softmax
- Regression: Linear (no activation)

✅ **Which optimizer to use:**
- Default: Adam (lr=0.001)
- Transformers/GPT: Adam
- Classic vision: SGD+Momentum
- **When in doubt: Adam!**

✅ **How to train:**
- Split data: 80% train, 10% val, 10% test
- Use batches (16-128)
- Monitor validation loss
- Early stopping when no improvement
- Use Adam optimizer

### Connection to GPT
✅ **GPT uses the EXACT SAME concepts:**
- Feed-forward layers (Module 3)
- GELU activation (Lesson 2)
- Backpropagation (Lesson 4)
- Adam optimizer (Lesson 6)
- Mini-batch training (Lesson 5)

**The only new concept in GPT is attention (Module 4)!**
