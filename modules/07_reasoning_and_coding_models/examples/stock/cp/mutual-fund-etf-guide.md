## PART E: SIP vs Lump Sum — Investment Strategy {#part-e}

### When to Use SIP vs Lump Sum

```
┌─────────────────────────────────────────────────────────────────────────┐
│  STRATEGY    │  WHEN TO USE                │  BEST FOR                  │
├──────────────┼─────────────────────────────┼────────────────────────────┤
│  Regular SIP │  Monthly salary →           │  Most investors (default)  │
│  (Fixed)     │  fixed amount every month   │  Removes timing worry      │
├──────────────┼─────────────────────────────┼────────────────────────────┤
│  Step-Up SIP │  Increase SIP by 10-15%     │  Salary grows annually →   │
│              │  every year automatically    │  investment should too     │
├──────────────┼─────────────────────────────┼────────────────────────────┤
│  Tactical    │  Normal SIP + extra lump    │  During market crashes     │
│  SIP         │  sums during big dips       │  (like NOW in March 2026)  │
├──────────────┼─────────────────────────────┼────────────────────────────┤
│  Lump Sum    │  Invest entire amount at    │  When you're SURE market   │
│              │  once                       │  is near bottom (risky!)   │
├──────────────┼─────────────────────────────┼────────────────────────────┤
│  STP         │  Park lump sum in liquid    │  Best of both worlds —     │
│  (Systematic │  fund → auto-transfer to    │  safety + gradual equity   │
│  Transfer    │  equity fund monthly        │  entry over 3-6 months     │
│  Plan)       │                             │                            │
└──────────────┴─────────────────────────────┴────────────────────────────┘
```

### ⭐ Recommended Strategy for Market Downturn (March 2026)

```
BEST APPROACH: "CORE SIP + TACTICAL TOP-UP"

Step 1: Start regular monthly SIP (your core investment)
        → Runs every month regardless of market

Step 2: Keep 20-30% of your investable amount as "dip money"
        → When Nifty falls another 5-10%, invest a chunk as lump sum

Step 3: Enable Step-Up SIP (increase by 10% annually)
        → As your income grows, your investments grow too

Example with ₹5,000/month budget:
  → Regular SIP: ₹3,500/month (runs every month)
  → Dip money: ₹1,500/month saved aside
  → When market dips 5%+: Deploy accumulated dip money as lump sum
```

## PART F: How to Actually Invest — Step-by-Step {#part-f}

### For Mutual Funds (SIP & Lump Sum)

```
Step 1: Choose a Platform
        ├── Groww (easiest for beginners)
        ├── Zerodha Coin (if you already have Zerodha demat)
        ├── ET Money (great comparison tools)
        ├── Kuvera (100% free, no commission)
        └── Paytm Money (convenient for Paytm users)

Step 2: Complete KYC
        → Aadhaar + PAN + Bank account
        → Takes 5-10 minutes online (video KYC)

Step 3: Search the fund
        → Search by name → Select "Direct - Growth" plan
        → ⚠️ ALWAYS select "Direct" and "Growth" option

Step 4A: For SIP
        → Click "Start SIP"
        → Enter monthly amount (min ₹100-500)
        → Choose SIP date (any date works, 1st or 15th common)
        → Set up auto-debit from bank
        → Done! It runs automatically every month.

Step 4B: For Lump Sum
        → Click "Invest Now" or "One-Time"
        → Enter amount
        → Pay via UPI/Net Banking
        → Units allotted next business day

Step 5: Enable Step-Up SIP (optional but recommended)
        → Some platforms allow automatic annual increase
        → Set 10% increase per year
```

### For ETFs

```
Step 1: You NEED a Demat + Trading Account
        ├── Zerodha (most popular, ₹0 delivery charges)
        ├── Groww
        └── Any SEBI-registered broker

Step 2: Search the ETF by name
        → e.g., "NIFTYBEES" or "GOLDBEES" or "MON100"

Step 3: Buy like a stock
        → Place a market order or limit order
        → Units credited to your demat account

Step 4: For regular investing in ETFs
        → Set a monthly reminder to manually buy
        → (ETFs don't support auto-SIP on most platforms)
        → Alternative: Use "ETF Fund of Fund" mutual fund for SIP
```

### How to Use Value Research to Pick a Fund (Step-by-Step)
```
1. Go to valueresearchonline.com
2. Click "Fund Selector" → Choose category (e.g., "Flexi Cap")
3. Sort by:
   a. Rating: ★★★★★ or ★★★★ (4-5 star funds only)
   b. 5-Year Return: Highest first
   c. Expense Ratio: Lowest preferred
4. Click on a fund → Check:
   a. "Returns" tab → Compare with benchmark & category average
   b. "Risk" tab → Check Sharpe Ratio (higher = better)
   c. "Portfolio" tab → See what stocks/bonds the fund holds
   d. "Fund Manager" tab → Check experience & track record
5. Shortlist 2-3 funds
6. Compare side-by-side using "Compare Funds" tool
7. Pick the one with: Best 5Y rolling returns + Lowest expense + Top rating
```

### How to Use ET Money Screener
```
1. Go to etmoney.com/mutual-funds/all-funds-listing
2. Filter by:
   a. Category → Select (e.g., "Large Cap")
   b. Plan → "Direct" (ALWAYS)
   c. Rating → 4★ or 5★
3. Sort by 5-Year returns
4. Click on fund → Review:
   a. Return vs Benchmark chart
   b. Peer comparison table
   c. Risk analysis (Standard Deviation, Sharpe Ratio)
5. Click "Compare" to compare up to 4 funds side-by-side
```

### How to Use fundscreener.in (Power User)
```
1. Go to fundscreener.in
2. Set filters:
   a. Fund Type: Equity
   b. Sub-Category: (your choice)
   c. Plan: Direct
   d. Returns 5Y: > 15%
   e. Expense Ratio: < 1.0%
   f. AUM: > 500 Cr
   g. Fund Age: > 5 years
3. Click "Screen" → Export results to Excel
4. Sort and analyze at your convenience
```
