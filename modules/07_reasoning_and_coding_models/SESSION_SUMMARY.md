## 🔗 Integration Guide

### Using Lesson 4 & 5 (Reasoning)

```python
# Import components
from example_04_process_supervision import ProcessRewardModel
from example_05_reasoning_system import O1ReasoningSystem

# Your GPT model from Module 6
your_gpt = YourGPTModel(...)

# Build o1-style system
prm = ProcessRewardModel()
o1_system = O1ReasoningSystem(
    base_model=your_gpt,
    process_reward_model=prm,
    max_thinking_steps=100,
    beam_width=5
)

# Solve with verified reasoning
result = o1_system.solve("Complex problem here")
print(f"Answer: {result['answer']}")
print(f"Confidence: {result['confidence']:.0%}")
```
